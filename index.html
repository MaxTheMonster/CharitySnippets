Buttons

Buttons are an essential part of every website, from taking user to the next page, to purchasing an item. Here are a few basic buttons that you may need to include in your website.


  1. HTML 

    
  <input type="button" name="button-1" value="Button 1" class="button-1">

    CSS 

    
  .button-1 {
    background-color: #4CAF50;
    border: none;
    color: white;
    padding: 1em;
    text-align: center;
    text-decoration: none;
    display: inline-block;
    font-size: 16px;
    border-radius: 2px;
    cursor: pointer;
    outline: 0;
  }

  2. HTML 

    
  <input type="button" name="button-2" value="Button 2" class="button-2">

    CSS 

    
  .button-2 {
    background-color: #0040FF;
    border: none;
    color: white;
    padding: 1em;
    text-align: center;
    text-decoration: none;
    border-radius: 5px;
    cursor: pointer;
    border: 3px solid # 002699;
    display: inline-block;
    font-size: 16px;
    outline: 0;
  }

  .button-2:hover {
    background: #002699;
  }

Search Bars

A good search bar can mean the difference between if a user gets to their page or not and whether you get the donation or not. These are essential to an easily navigable website as they are arguably superior to other forms of navigation such as a drop down as they provide the user a nearly infinite option set of what they can look for.


  1. HTML 

    
  <input type="text" name="search-1" placeholder="Search..." class="search-1">

    CSS 

    
  .search-1 {
    outline: 0;
    border-radius: 5px;
    border: 1px solid rgba(0,0,0,0.5);
    width: 30%;
    font-size: 2rem;
    padding: 0.4em;
  }

  2. HTML 

    
  <input type="text" name="search-2" placeholder="Search..." class="search-2">

    CSS 

    
  .search-2 {
    outline: 0;
    font-size: 1.66rem;
    padding: 0.2em;
    border-bottom: 5px solid rgba(0,0,0,0.5);
    transition: 0.4s;
  }
  .search-2:focus {
    border-bottom: 5px solid #d35400;
    transition: 0.2s;
  }

  3. HTML 

    
  <input type="text" name="search-3" placeholder="Search..." class="search-3">

    CSS 

    
  .search-3 {
    font-family: sans-serif;
    padding: 0.5em;
    outline: 0;
    font-size: 2rem;
  }

HTML Starter

There are many ways to do it, but many believe that this is the best way to start an HTML document - it is also the simplest. Just paste this into your code and you are up and running to get coding.


  1. HTML 

    
  <!DOCTYPE html>
  <html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta author="John Appleseed">
    <meta description="A website about how much I love apples.">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Apples are awesome!</title>
  </head>
  <body>
    <h1>I love apples!</h1>
  </body>
  </html>

Video

Videos seem to be the new way that people are telling stories and consuming information and with the rise of platforms such as YouTube, video is one of the best ways to communicate your ideas. This example contains a fallback for older browsers that do not support the <video> tag and for .ogg and .mp4 supporting browsers.


  1. HTML 

    
  <video width="360" height="240" controls>
    <source src="img/video.mp4" type="video/mp4">
    <source src="movie.ogg" type="video/ogg">
    Your browser does not support video.
  </video>

    CSS 

    
  .video-1 {
    background: rgba(0,0,0,0.5);
  }
